Visual Studioで、Prismの「Xamarin Forms」を利用していた際、UWPプロジェクトのビルドが通らなくてエラーが発生しました。  
忘れないように対処方法を残しておきます。  

## 現象

環境は下記

- Windows 10
- Visual Studio 2022
- Prism Template Pack インストール済み

エラーメッセージはこちらです。  

![image](/blog/33/1.png)  

> .NETCore,Version=v5.0 の参照アセンブリが見つかりませんでした。この問題を解決するには、このフレームワーク バージョンの Developer Pack (SDK/Targeting Pack) をインストールするか、アプリケーションのターゲットを再設定してください。[https://aka.ms/msbuild/developerpacks](https://aka.ms/msbuild/developerpacks) で .NET Framework Developer Pack をダウンロードできます。


## 対処

Visual Studio インストーラーから、**ユニバーサル Windows プラットフォーム開発をインストール**する必要があります。  

![image](/blog/33/3.png)  

検証できていませんが、「.NET マルチプラットフォーム アプリの UI開発」のインストールも必要だった可能性があります。  
（今回のエラーが発生した際、マルチプラットフォームUI開発は既にインストール済みだったため）
